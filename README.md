# Pannenspel Scorebord

Een simpel scorebord voor het pannenspel (Beschrijven / Eén woord / Uitbeelden).
Statische pagina, geen build-stap nodig — werkt direct via GitHub Pages.

## Gebruik

1. Open de pagina en voeg teams toe (naam + kleur), volgorde van toevoegen = beurtvolgorde.
2. Klik "Spel starten".
3. Tik op het scherm om een punt toe te kennen aan het team dat aan de beurt is.
4. Na 1 minuut stopt de klok automatisch; de volgende tik start de volgende beurt.
5. Zegt een speler een woord dat op de kaart staat? Klik "Fout (stop beurt)" — de beurt
   stopt meteen, maar de punten van die beurt blijven staan.
6. Is de pot leeg? Druk op de spatiebalk om naar de volgende fase te gaan. Was een team
   nog bezig, dan gaat de klok in de volgende fase verder met de resterende tijd.
7. Na de derde fase (Uitbeelden) verschijnt de eindstand.

## Hosten via GitHub Pages

Instellingen → Pages → "Deploy from a branch" → branch `main`, map `/ (root)`.
